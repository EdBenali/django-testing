Welcome to your Django sandbox

This project is set up for hands-on practice. Start with the pages app, then add models, forms, and auth as you work through the topics below.

Suggested learning path

  1. URL routing and views — edit pages/views.py and pages/urls.py
  2. Templates and context — extend templates/base.html
  3. Models and migrations — add fields in pages/models.py, run migrate
  4. Django admin — register models in pages/admin.py
  5. Forms and validation — create a simple contact or note form
  6. Authentication — use built-in User model and login views
  7. Class-based views — refactor function views to CBVs
  8. Testing — write tests in pages/tests.py
